# Crud NestJs Prisma Postgresql Passport Jwt

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## To make husky work on (Mac/Linux) you need to run in the root

```bash

chmod ug+x .husky/*
chmod ug+x .git/hooks/*
```
